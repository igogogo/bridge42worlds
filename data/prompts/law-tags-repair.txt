Закон/принцип/теорема ниже сейчас связан ТОЛЬКО с образовательными тегами (они не входят в промт генерации статей) — из-за этого у него НИКОГДА не появится ни одной «похожей статьи» на сайте, хотя по сути он вполне может пересекаться с реальным контентом.

Закон: $law_ru (тип: $law_type)
Текущие теги (образовательные, оставь их как есть — не убирай): $current_tags

Подбери 1-3 id из списка АКТИВНЫХ тегов ниже, которые РЕАЛЬНО относятся к этому закону по смыслу. Не притягивай за уши — если закон и правда не пересекается по сути ни с одним активным тегом, верни пустой список.

Доступные id активных тегов (используй ТОЛЬКО их):
$tag_ids

Ответь JSON: {"tags": ["id1", "id2"]}
