Ниже — список тегов, которые пока НЕ связаны ни с одним законом и ни с одним учёным в базе
(это разрывает граф знаний — на странице тега нет связанных законов/учёных, и наоборот).

Для КАЖДОГО тега подбери ОДИН наиболее подходящий существующий закон ИЛИ ОДНОГО существующего
учёного из списков ниже, которого честно стоит связать с этим тегом. Если для тега реально нет
подходящего закона/учёного среди перечисленных — верни null для этого тега, не притягивай за уши
случайную связь.

Теги без связи (id):
{tags_json}

Существующие законы (id: русское название):
{laws_json}

Существующие учёные (имя):
{scientists_json}

Ответь JSON-объектом с полем "links" — по одной записи на КАЖДЫЙ тег из списка выше:
{{"links": [{{"tag": "...", "target_type": "law", "target_id": "..."}}, {{"tag": "...", "target_type": "scientist", "target_id": "..."}}, {{"tag": "...", "target_type": null, "target_id": null}}]}}
