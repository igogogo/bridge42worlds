$focus_line
Составь список из $need учёных. Для каждого укажи:
- id: имя на английском (оригинал, не переводи) — это же отображаемое имя ВЕЗДЕ и на всех языках
- lifespan: годы жизни
- description: развёрнутое описание вклада
- biography: краткая биография — где учился, где работал, с кем, награды
- key_discoveries: ключевые открытия с пояснениями (на русском)
- fields: области науки (на русском)
- related_tags: основные английские id ТОЛЬКО из этого списка: $tags_str
- quote: известные цитаты (на русском)
- fun_fact: интересные факты (на русском)

Ответь JSON-объектом:
{"scientists": [{"id": "Isaac Newton", ...}]}
