String Theory

📊 0 articles
A hypothetical theory in which fundamental particles are not points but one-dimensional vibrating strings. Different vibration frequencies of the string give rise to different particles. Imagine that all particles are notes played by a violinist on a single string: the melody depends on how the string vibrates.
💡 String theory predicts the existence of the graviton — a particle that mediates gravity, making it a candidate for a theory of everything.

History

String theory originated in the late 1960s to describe the strong interaction. In the 1980s, after the first superstring revolution, it became clear that it could unify all fundamental interactions. The second revolution in the 1990s led to M-theory.

How it works

Strings have a size on the order of the Planck length (10^-35 m). They can be open (free ends) or closed (loops). String vibrations determine the mass and charge of a particle. Analogy: a guitar string — different ways of plucking produce different sounds.

Open problems & fun facts

String theory has no experimental confirmation, as it requires energies inaccessible to current accelerators. Moreover, it predicts many extra dimensions that are not observed.
