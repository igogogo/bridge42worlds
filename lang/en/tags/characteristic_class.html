Characteristic class

📊 0 articles
It's like a 'barcode' for bundles: from it you can determine how 'twisted' the space over the base is.
💡 Characteristic classes help understand why you can't comb a hedgehog on a sphere — there is a topological obstruction.
Characteristic classes are cohomological invariants of vector bundles that measure their 'twistedness'. For example, the Euler class shows whether there is a nonzero section, and the Pontryagin and Chern classes give information about curvature.

History

The concept was introduced by Hassler Whitney and Eduard Stiefel in the 1930s, then developed by Shiing-Shen Chern and Lev Pontryagin.

How it works

Each vector bundle is assigned cohomology classes that are computed via the curvature of a connection (for smooth bundles) or via obstructions to constructing independent sections.

💡 Chern classes are used in physics to describe topological phases of matter, for example, in the quantum Hall effect.
Characteristic classes are cohomology classes associated with vector bundles that are invariant under bundle isomorphisms. Main types: Stiefel-Whitney classes (w_i, Z/2-cohomology), Pontryagin classes (p_i, Z-cohomology), and Chern classes (c_i, Z-cohomology for complex bundles). They satisfy axioms: functoriality, normalization, Whitney sum. The Euler class is the top Stiefel-Whitney class for oriented bundles.

History

Hassler Whitney in 1935 introduced classes named after him for studying spheres. Eduard Stiefel independently obtained similar results. Shiing-Shen Chern in 1946 defined classes for complex bundles. Lev Pontryagin in 1947 introduced classes for real bundles. In the 1950s, Friedrich Hirzebruch used them to prove the Riemann-Roch theorem.

How it works

For a smooth vector bundle, a connection is chosen; its curvature is a 2-form with values in endomorphisms. Symmetric polynomials in the curvature give closed forms whose cohomology classes are independent of the connection. For example, the first Chern class c_1 = [i/(2π) Tr(F)], where F is the curvature. These classes are obstructions to trivializing the bundle.

c_1 = \frac{i}{2\pi} \operatorname{Tr}(F)
The Chern class is expressed via the curvature of a connection F as the cohomology class of the trace of curvature.

Open problems & fun facts

Computing characteristic classes for exotic bundles; Connection with topological field theory and quantum invariants

💡 Chern classes are used in physics to describe topological phases of matter, for example, in the quantum Hall effect.
