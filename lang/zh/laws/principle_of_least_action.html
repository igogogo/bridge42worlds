Принцип наименьшего действияпринцип

Природа «ленива»: из всех возможных путей система выбирает тот, который требует наименьшего действия — как вода, которая течёт по самому короткому пути вниз.
💡 Этот принцип работает для света (закон Снеллиуса), для планет (орбиты) и даже для квантовых частиц (интегралы по траекториям).
Принцип наименьшего действия утверждает, что из всех возможных траекторий движения система выбирает ту, для которой действие (интеграл от лагранжиана по времени) минимально или стационарно. Математически: δS = 0, где S = ∫L dt. Это общий принцип, из которого следуют уравнения движения (например, уравнения Ньютона).

工作原理

Принцип используется для вывода уравнений движения в механике, электродинамике, теории поля. Например, траектория мяча, брошенного под углом, — это путь с наименьшим действием. В квантовой механике принцип обобщается до интегралов по траекториям Фейнмана.

💡 Пьер де Ферма в 1662 году сформулировал принцип наименьшего времени для света, который стал предшественником принципа наименьшего действия.
Принцип наименьшего действия — вариационный принцип, согласно которому истинная траектория физической системы является стационарной точкой функционала действия S = ∫L dt, где L — лагранжиан. Для консервативных систем это минимум действия. Из принципа следуют уравнения Эйлера-Лагранжа, которые эквивалентны уравнениям Ньютона. Принцип обобщается на поля и квантовую механику.

发现历史

Принцип восходит к Пьеру де Ферма (1662) для оптики. В 1744 году Пьер-Луи Мопертюи сформулировал принцип наименьшего действия для механики. Леонард Эйлер и Жозеф Луи Лагранж развили математический аппарат (вариационное исчисление). В 1834 году Уильям Гамильтон дал современную формулировку через лагранжиан. Ричард Фейнман в 1948 году обобщил принцип на квантовую механику (интегралы по траекториям).

工作原理

Принцип используется для вывода уравнений движения в классической механике, теории поля, электродинамике, общей теории относительности. Он лежит в основе лагранжева и гамильтонова формализма. Границы применимости: принцип работает для консервативных и диссипативных систем (с обобщёнными силами). В квантовой механике заменяется интегралом по траекториям.

注意事项

Принцип не даёт объяснения, почему природа «выбирает» экстремум — это аксиома.; Для систем с трением требуется модификация (принцип наименьшего действия не работает напрямую).

S = \int_{t_1}^{t_2} L(q,\dot{q},t) dt
Функционал действия, где L — лагранжиан, q — обобщённые координаты.
\frac{d}{dt} \frac{\partial L}{\partial \dot{q}} - \frac{\partial L}{\partial q} = 0
Условие стационарности действия, дающее уравнения движения.
