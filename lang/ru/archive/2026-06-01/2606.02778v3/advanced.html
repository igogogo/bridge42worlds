ExoVeil: Обнаружение экзопланет по одному транзиту через самосверку

Original: "One Transit Is All You Need: Detecting Exoplanets Through Learned Stellar Behaviour with EXOVEIL"
Система ExoVeil учится предсказывать блеск звезды и находит одиночные транзиты, которые пропускают классификаторы.

Контекст

Современные методы машинного обучения для поиска транзитов, такие как ExoMiner и AstroNet, требуют известного орбитального периода и фазового сворачивания. Это делает их бесполезными для планет с одним транзитом, которые являются приоритетными для миссий вроде PLATO. Экзопланеты с долгими периодами — основная цель, но структурный недостаток пайплайнов не позволяет их детектировать.

Методы

Автор обучил трансформерную мировую модель на 16499 кривых блеска Kepler с самоконтролируемым обучением, маскируя известные транзиты. Модель предсказывает ожидаемый поток, а матчевый фильтр с весами по дисперсии извлекает сигналы из остатков. Затем XGBoost-классификатор на 21 признаке отделяет планеты от ложных срабатываний. Применено конформное предсказание для оценки неопределенности.

Результаты

Слепой поиск по 3737 звёздам Kepler выявил 179 транзитоподобных аномалий, не входящих в каталог DR25. При инжекции одиночных транзитов система восстанавливает 32% на глубине 1000 ppm, в то время как классификаторы дают 0% по построению. При нулевом переносе на миссию TESS восстановлены все 47 подтверждённых планет в поле PLATO LOPS2 (100% recovery). При каденции PLATO 25 секунд чувствительность достигает 100 ppm, приближаясь к аналогам Земли.

Значение

Результаты показывают, что парадигма предсказания вместо классификации может заполнить пробел в обнаружении экзопланет с малым числом транзитов. Звёздный блеск моделируется с помощью обучения, что открывает путь к обнаружению планет земного типа у телескопов нового поколения.

Развитие

Планируется замена трансформера на линейную архитектуру (Mamba) для обработки полных кривых без окон. Добавление автоматического отклонения четырёх классов ложных срабатываний (послевспышечный выброс, ошибка слежения за вращением и др.). Расширение на мультиобъективную миссию PLATO с 24 камерами.

Влияние

Повлияет на пайплайны обработки данных миссий PLATO и других обзоров, а также на стратегию отбора кандидатов для наземного подтверждения.

Следующие шаги

Провести крупномасштабные инжекции с PlatoSim, включить анализ центроида и спектроскопию для верификации кандидатов.

Ключевые проблемы

Работа напрямую связана с проблемой обнаружения экзопланет земного типа вокруг солнцеподобных звёзд, которая является центральной для миссии PLATO и разрешается при помощи методов машинного обучения.

🎯 Система ExoVeil впервые применяет конформное предсказание к ветированию транзитов, обеспечивая формальную гарантию покрытия 95.9%.

Ключевые числа

  • tess_loops2_recovery: 100% (47/47)
  • blind_search_anomalies: 179
  • conformal_coverage: 95.9%
  • plato_sensitivity: 100 ppm
