Интеграл по траекториям

📊 0 articles
Интеграл по траекториям — это метод квантовой физики, который суммирует все возможные пути частицы из точки А в точку Б, как если бы частица «пробовала» все маршруты одновременно.
💡 Ричард Фейнман придумал этот метод, глядя на рябь на воде: волны распространяются по всем возможным путям.
Интеграл по траекториям — это способ вычисления вероятности перехода квантовой системы из одного состояния в другое путём суммирования по всем возможным путям (траекториям) с весом, определяемым действием. Каждый путь вносит вклад exp(iS/ħ), где S — классическое действие. В классическом пределе ħ→0 доминирует путь, минимизирующий действие (принцип наименьшего действия).

История

Метод был разработан Ричардом Фейнманом в 1948 году на основе идей Поля Дирака. Фейнман использовал его для переформулировки квантовой механики и квантовой электродинамики, за что получил Нобелевскую премию в 1965 году.

Как работает

Представьте, что вы хотите перейти из одного города в другой. Интеграл по траекториям говорит: нужно рассмотреть все возможные маршруты — по прямой, с объездами, петлями — и сложить их вклады, учитывая фазу. В результате получается амплитуда вероятности. На практике суммирование бесконечного числа путей выполняется аналитически с помощью методов функционального анализа.

💡 Интеграл по траекториям используется не только в квантовой механике, но и в финансовой математике для оценки опционов (модель Блэка-Шоулза).
Интеграл по траекториям (фейнмановский интеграл) — это функциональный интеграл, представляющий амплитуду перехода как сумму по всем путям в конфигурационном или фазовом пространстве. Формально: ⟨x_f|e^{-iHt/ħ}|x_i⟩ = ∫ Dx(t) exp(iS[x]/ħ), где S[x] — действие. Интеграл по траекториям является основой квантовой теории поля, где он обобщается на поля. Он позволяет вычислять корреляционные функции и амплитуды рассеяния. Метод требует регуляризации и перенормировки для устранения расходимостей.

История

Идея интеграла по траекториям восходит к Полю Дираку (1933), который заметил, что квантовая амплитуда связана с классическим действием. Ричард Фейнман в 1948 году развил полную формулировку, опубликовав статью «Пространственно-временной подход к квантовой электродинамике». Фейнман использовал диаграммы для визуализации вкладов. Метод стал ключевым в квантовой теории поля и статистической механике. За разработку квантовой электродинамики Фейнман получил Нобелевскую премию в 1965 году.

Как работает

Интеграл по траекториям определяется как предел дискретизации времени: разбиение временного интервала на N шагов, вставка полноты на каждом шаге, затем переход к непрерывному пределу. Для свободной частицы интеграл гауссов и вычисляется точно. Для взаимодействующих полей используются методы теории возмущений (диаграммы Фейнмана). Интеграл по траекториям также применяется в евклидовой формулировке (мнимое время) для вычисления статистической суммы.

\langle x_f | e^{-iHt/\hbar} | x_i \rangle = \int \mathcal{D}x(t) \, e^{iS[x]/\hbar}
Амплитуда вероятности перехода из x_i в x_f за время t равна интегралу по всем путям x(t) с весом exp(iS/ħ).

Открытые проблемы и интересные факты

Строгое математическое определение интеграла по траекториям (функциональный интеграл не является мерой в обычном смысле).; Проблема непертурбативных методов: как вычислять интегралы для сильно взаимодействующих теорий (например, КХД).

💡 Интеграл по траекториям используется не только в квантовой механике, но и в финансовой математике для оценки опционов (модель Блэка-Шоулза).
