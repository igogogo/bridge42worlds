Энтропия Шеннона

📊 0 articles
Энтропия Шеннона — это мера неопределённости: чем больше возможных вариантов, тем выше энтропия. Например, у честной монеты энтропия 1 бит, а у монеты с двумя орлами — 0.
💡 Энтропия Шеннона совпадает по формуле с энтропией Больцмана в статистической физике.
Энтропия Шеннона — это среднее количество информации, которое несёт один символ сообщения. Для случайной величины X с распределением p(x) она равна H(X) = -∑ p(x) log p(x). Чем более равномерно распределение, тем выше энтропия.

История

Введена Клодом Шенноном в 1948 году как мера информации. Название «энтропия» предложил Джон фон Нейман из-за сходства с термодинамической энтропией.

Как работает

Если событие имеет вероятность p, то его информационное содержание равно -log p. Энтропия — среднее таких значений. Например, для равномерного распределения с 8 исходами энтропия равна 3 битам.

💡 Энтропия Шеннона используется в машинном обучении для построения деревьев решений (критерий информационного выигрыша).
Энтропия Шеннона — фундаментальная мера неопределённости в теории информации. Для дискретной случайной величины X с множеством значений {x_i} и вероятностями p_i, энтропия H(X) = -∑ p_i log_b p_i, где основание логарифма определяет единицу (бит, нат, хартли). Свойства: неотрицательность, максимальна для равномерного распределения, аддитивность для независимых величин. Энтропия — частный случай энтропии Реньи. Она связана с минимальной длиной кода без потерь (теорема Шеннона о кодировании источника).

История

Клод Шеннон ввёл энтропию в 1948 году. Джон фон Нейман посоветовал название «энтропия», так как она напоминает формулу Больцмана S = k log W. В 1950-х энтропия стала ключевым понятием в теории кодирования. Позже обобщена на непрерывные распределения (дифференциальная энтропия) и квантовые системы (энтропия фон Неймана).

Как работает

Энтропия вычисляется по распределению вероятностей. Для дискретного случая H = -∑ p_i log_2 p_i (биты). Для непрерывного случая — дифференциальная энтропия h(X) = -∫ f(x) log f(x) dx. Энтропия может быть условной H(Y|X) и совместной H(X,Y). Взаимная информация I(X;Y) = H(X) - H(X|Y) = H(Y) - H(Y|X).

H(X) = -\sum_{i=1}^{n} p_i \log_2 p_i
Среднее количество информации (в битах) на символ для источника с распределением p_i.

Открытые проблемы и интересные факты
