Характеристический класс

📊 0 articles
Это как «штрих-код» для расслоений: по нему можно определить, насколько «закручено» пространство над базой.
💡 Характеристические классы помогают понять, почему на сфере нельзя причесать ёжика — есть топологическое препятствие.
Характеристические классы — это когомологические инварианты векторных расслоений, которые измеряют их «скрученность». Например, класс Эйлера показывает, есть ли ненулевое сечение, а классы Понтрягина и Черна дают информацию о кривизне.

История

Понятие введено Хаслером Уитни и Эдуардом Штифелем в 1930-х, затем развито Шин-Шен Черном и Львом Понтрягиным.

Как работает

Каждому векторному расслоению сопоставляются когомологические классы, которые вычисляются через кривизну связности (для гладких расслоений) или через препятствия к построению независимых сечений.

💡 Классы Черна используются в физике для описания топологических фаз материи, например, в квантовом эффекте Холла.
Характеристические классы — это когомологические классы, ассоциированные с векторными расслоениями, которые инвариантны относительно изоморфизмов расслоений. Основные типы: классы Штифеля-Уитни (w_i, Z/2-когомологии), классы Понтрягина (p_i, Z-когомологии) и классы Черна (c_i, Z-когомологии для комплексных расслоений). Они удовлетворяют аксиомам: функториальность, нормализация, сумма Уитни. Класс Эйлера — это класс Штифеля-Уитни старшей размерности для ориентированных расслоений.

История

Хаслер Уитни в 1935 году ввёл классы, названные его именем, для изучения сфер. Эдуард Штифель независимо получил аналогичные результаты. Шин-Шен Черн в 1946 году определил классы для комплексных расслоений. Лев Понтрягин в 1947 году ввёл классы для вещественных расслоений. В 1950-х Фридрих Хирцебрух использовал их для доказательства теоремы Римана-Роха.

Как работает

Для гладкого векторного расслоения выбирается связность, её кривизна — 2-форма со значениями в эндоморфизмах. Симметрические полиномы от кривизны дают замкнутые формы, когомологические классы которых не зависят от выбора связности. Например, первый класс Черна c_1 = [i/(2π) Tr(F)], где F — кривизна. Эти классы являются препятствиями к тривиализации расслоения.

c_1 = \frac{i}{2\pi} \operatorname{Tr}(F)
Класс Черна выражается через кривизну связности F как когомологический класс следа кривизны.

Открытые проблемы и интересные факты

Вычисление характеристических классов для экзотических расслоений; Связь с топологической теорией поля и квантовыми инвариантами

💡 Классы Черна используются в физике для описания топологических фаз материи, например, в квантовом эффекте Холла.
